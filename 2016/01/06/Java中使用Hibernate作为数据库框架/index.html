1.下载hibernate jar文件 http://7xpas5.com1.z0.glb.clouddn.com/hibernate.zip

2.下载mysql连接 jar文件 http://7xpas5.com1.z0.glb.clouddn.com/mysql.zip

3.配置好mysql的权限

你的帐号不允许从远程登陆，只能在localhost。这个时候只要在localhost的那台电脑，登入mysql后，更改 “mysql” 数据库里的 “user” 表里的 “host” 项，从”localhost”改称”%”

$mysql -u root -p*

mysql>update user set host = ‘%’ where user = ‘root’;

mysql>select host, user from user;

引用 描述
“$ref”:”..” 上一级
“$ref”:”@” 当前对象，也就是自引用
“$ref”:”$” 根对象
“$ref”:”$.children.0” 基于路径的引用，相当于 root.getChildren().get(0)
