C++命名空间

为了避免命名冲突，c++ 引入命名空间

若把两个变量放入一个函数体里面使用，则需要指明是哪个命名空间里的,使用域解析操作符”::”操作符

int max(a,b){
}

main(){
int (*mMax)();
mMax = max;

int i = (*mMax)(a,b);
}
```
